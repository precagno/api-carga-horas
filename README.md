## Carga de horas docentes

Api ficticia desarrollada con Spring Boot hecha para carga de horas docentes de Villa Ballester.

La API cuenta con los siguientes endpoints: 
	
### Horas
Verb | URL |
|---|---|
| GET 	 | /api/horas		|
| GET 	 | /api/horas/{id}	|
| POST 	 | /api/horas		|
| PUT 	 | /api/horas/{id}	|
| DELETE | /api/horas/{id}	|
